FlexBox: Flex Direction

The flex-direction property specifies the direction of the flexible items. By default, the flex items are laid out in a row. It can be confusing because to us they appear as columns.

There will be times that you would want it to be set to column and still take advantage of flex-direction: column

Flex Basis

The flex-basis property specifies the initial main size of a flex item. It sets the size of the content box unless otherwise set with box-sizing.

You will see that it does matter if you set the flex-direction to row or column.

It's really important to know the difference between the Parent Element vs the Child Elements. And what property goes where. Somethings go on the Parent Container and others will go on the Child Items.

Flex Order

The order property specifies the order of a flexible item relative to the rest of the flex items inside the same container. This property should go on the Parent

Justify Content

The justify-content property aligns the flexible container's items when the items do not use all available space on the main-axis.

Possible Values
  • flex-start
  • flex-end
  • center
  • space-between
  • space-around

Align Items

The align-items property specifies the default alignment for items inside the flexible container.

To see this take effect we must provide a height or width to the cross axis.

Possible Values
  • stretch
  • flex-start
  • flex-end
  • center
  • baseline
